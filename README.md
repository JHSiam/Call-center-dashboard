# Call Center Dashboard

A modern, responsive Call Center Dashboard built using React and Tailwind CSS.  
This project provides a clean and intuitive interface for managing call logs, appointments, and user settings.

## Features

- Fully responsive dashboard layout
- Sidebar navigation
- Dashboard overview page
- Call logs page
- Appointments page
- Settings page (Profile & Password)
- Modern dark UI
- Reusable React components

## Getting Started

Follow the steps below to run the project locally from GitHub.

### Prerequisites

Make sure you have the following installed:

- Node.js (v16 or later)
- npm or yarn

```bash

2. Go to the Project Directory
